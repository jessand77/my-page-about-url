URL

URL stands for Uniform Resource Locator. It is basically the address of a resource on the Web. Examples of resources are HTML pages, CSS documents and images. When a URL is typed into the address bar of a browser, the browser will load the resource (at least if it's an existing URL). Typically a URL consists of a protocol (or scheme), a domain name and a file name. URLs are also used by other tecnologies, for example in html files where they are used to create links or display images and videos, among other things.
